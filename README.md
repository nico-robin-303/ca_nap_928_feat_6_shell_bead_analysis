# ca_nap_928_feat_6_olivella_cluster_analysis
olivella shell beads and detritus from a pre-contact house floor feature in napa, ca, mapped in a spatial cluster analysis.
this project is part of a an archaeological thesis for graduate studies in cultural resourse management at nmhu.
recommended citation: Sippola, Gunnar K. (2026) Napa Beads: An Interpretation of a Pre-Contact House Floor Feature through Recovered Olivella Shell Beads from CA-NAP-928. Master's Thesis. New Mexico Highlands University, Department of Sociology, Anthropology, and Criminal Justice. Las Vegas, New Mexico.
